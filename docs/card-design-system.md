## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **서비스명** | 운명의 아이 (Destiny Baby) |
| **URL** | destiny-baby.vercel.app |
| **서비스 설명** | 아기 사주(四柱)를 기반으로 한 AI 작명 및 탄생일 추천 서비스 |
| **핵심 기능** | 사주 분석 → 이름 추천 → 카드 뽑기 → 컬렉션 |
| **카드 시스템** | 포켓몬 카드 스타일의 수집형 운세 카드 (가챠 시스템) |
| **목표 오픈일** | 2025년 4월 |
| **운영 주체** | 한국 스타트업 (모바일 웹, 추후 앱 확장) |

### 서비스 플로우 요약
```
부모 사주 입력 → 아기 사주 분석 → 이름 추천 → [카드 뽑기] → 운명 카드 획득 → 컬렉션 저장 → 소셜 공유
```

카드는 단순한 UI 요소가 아닙니다. **서비스의 감동 포인트이자 공유 콘텐츠**입니다.
사용자가 카드를 받았을 때 "이거 친구한테 보내고 싶다"는 반응을 끌어내는 것이 핵심 목표입니다.

## 2. 서비스 성격 및 감성 방향

### 서비스 포지셔닝
- **장르**: 감성 + 엔터테인먼트 + 게임형 오락 AI 사주
- **포지션**: "진지한 운세 앱"이 아닌 "귀엽고 따뜻한 육아 게임"
- **레퍼런스 브랜드**: 사랑의 하츄핑, 포켓몬스터, 동물의 숲

### 감성 키워드 (우선순위 순)
1. **귀여움** — 가장 중요. 무조건 귀여워야 합니다
2. **따뜻함** — 아기, 엄마, 가족의 온기
3. **신비로움** — 사주/동양 철학의 마법적 분위기
4. **수집 욕구** — "하나 더 갖고 싶다"는 느낌
5. **정보 전달** — 오행, 등급, 능력치 (부차적)

### 톤 앤 매너
- 무겁지 않고 가볍고 유쾌함
- 어둡지 않고 따뜻하고 빛남
- 복잡하지 않고 직관적으로 귀여움
- 어른이 봐도 "귀엽다"고 느낄 수 있는 수준의 캐릭터화

## 3. 카드의 역할 및 사용자 감정 목표

### 카드가 하는 일
| 역할 | 설명 |
|------|------|
| **결과 전달** | 사주 분석 결과를 카드 형태로 시각화 |
| **수집 요소** | 총 180종 (MVP 30종 + AI 확장 150종) 수집 |
| **공유 콘텐츠** | 카카오톡/인스타그램 공유 — 카드 이미지 자체가 콘텐츠 |
| **가챠 보상** | 등급에 따른 희귀도 차등, 수집 욕구 자극 |

### 사용자가 카드를 받을 때 느껴야 하는 감정
1. **설렘** — "어떤 카드가 나올까?"
2. **애정/귀여움** — "아기 캐릭터가 너무 귀엽다"
3. **위로** — "내 아이의 운명이 이렇게 빛나는구나"
4. **재미** — "또 뽑고 싶다"
5. **수집 욕구** — "아직 못 모은 카드가 있다"

## 4. 플랫폼 및 카드 표시 환경

### 디스플레이 환경
- **주 플랫폼**: 모바일 웹 (iOS Safari, Android Chrome)
- **화면 크기**: 375px ~ 428px 기준 (iPhone SE ~ iPhone Pro Max)
- **배경 컬러**: 진한 남보라 그라디언트 `linear-gradient(135deg, #1A0A2E, #2D1B69)`
- **카드는 항상 어두운 배경 위에 표시됩니다**

### 카드 표시 크기 (3단계)
| 사이즈 | 픽셀 크기 | 사용 화면 |
|--------|-----------|-----------|
| **소 (sm)** | 96 × 135px | 컬렉션 그리드 (3열) |
| **중 (md)** | 160 × 228px | 목록, 카드 선택 |
| **대 (lg)** | 208 × 295px | 카드 뽑기 애니메이션, 상세 보기 |

**기준 사이즈: 208 × 295px (2:3 비율 세로형)**

### 카드 뽑기 애니메이션 (개발 완료)
카드는 뒷면 → 앞면으로 3D 플립 애니메이션됩니다.
- 등급에 따라 파티클 버스트 효과 발동
- 높은 등급일수록 화면 전체 플래시 효과 추가

## 5. 작업 범위 및 납품 수량

### MVP 납품 (1차)
- **총 30종**: 5오행 × 6등급 = 30개 일러스트
- 각 카드마다: 캐릭터 + 수호 동물/존재 + 배경 + 카드 프레임

### 확장 (2차, AI 생성)
- **총 150종 추가**: 직업 6종 × 5오행 × 5등급 (N~UR) 조합
- 1차 일러스트를 바탕으로 AI가 스타일 학습하여 생성
- **스타일 가이드 제공이 핵심** — 비율/색감/라인/배경 통일 필수

### 추가 요소
- **카드 뒷면 디자인**: 1종 (등급별 테두리 색상 변형)
- **스타일 가이드**: 비율, 컬러칩, 라인 두께, 음영 방식 문서화
- (옵션) 아티스트 콜라보 확장 시리즈 — 추후 논의

## 6. 카드 전체 구성 매트릭스

### 기본 구조
```
총 180종 = 6등급 (N/R/SR/SSR/UR/SSS) × 5오행 (木/火/土/金/水) × 6직업
MVP 30종 = 6등급 × 5오행 (직업 없음, 오행 정수 버전)
```

### MVP 30종 매트릭스

| 등급 | 木 (나무) | 火 (불) | 土 (흙) | 金 (금속) | 水 (물) |
|------|-----------|---------|---------|-----------|---------|
| **N** | 새싹의 기운 | 불꽃 씨앗 | 흙의 품 | 은빛 조각 | 이슬방울 |
| **R** | 초록의 힘 | 열정의 불꽃 | 대지의 힘 | 강철의 의지 | 흐르는 물 |
| **SR** | 숲의 수호자 | 태양의 빛 | 황금 들판 | 빛나는 보석 | 깊은 바다 |
| **SSR** | 봄바람의 축복 | 불사조의 깃털 | 산왕의 축복 | 백호의 포효 | 현무의 지혜 |
| **UR** | 세계수의 각성 | 태양신의 눈동자 | 대륙의 심장 | 천상의 검 | 용왕의 진주 |
| **SSS** | 창조의 숨결 | 영원의 불꽃 | 천지개벽 | 우주의 결정 | 시간의 파도 |

### 등급별 드롭 확률
| 등급 | 한국어명 | 드롭 확률 | 희귀도 |
|------|---------|-----------|--------|
| N | 일반 | 40% | ★ |
| R | 레어 | 30% | ★★ |
| SR | 슈퍼레어 | 15% | ★★★ |
| SSR | 초레어 | 8% | ★★★★ |
| UR | 울트라레어 | 4% | ★★★★★ |
| SSS | 신화 | 3% | ★★★★★★ |

**천장 시스템**: 50회 이내 SSR 이상 확정 / 90회 이내 SSS 확정

## 7. 카드 해부도 — 레이아웃 및 영역 규격

### 카드 치수
```
기준 캔버스: 208 × 295px (웹 표시 기준)
인쇄 대응: 744 × 1056px @ 300dpi (2:3 비율 동일)
모서리 반경: 12px (웹 기준), 43px (인쇄 기준)
```

### 카드 레이아웃 영역도

```
┌─────────────────────────────┐  ← 카드 외곽 테두리 (등급별 색상)
│  [오행아이콘]    [등급뱃지]  │  ← 상단 바 (높이 40px)
│   top-left       top-right  │
│                             │
│  ┌─────────────────────┐   │
│  │                     │   │
│  │                     │   │
│  │   [일러스트 영역]    │   │
│  │   캐릭터 + 수호동물  │   │
│  │   + 배경            │   │
│  │                     │   │
│  │    [중앙 아트]       │   │
│  │   (전체 높이의       │   │
│  │    약 65%)          │   │
│  │                     │   │
│  └─────────────────────┘   │
│                             │
│  ████████████████████████  │  ← 하단 정보 영역 (높이 70px)
│  [카드 이름 — 한국어]       │  ← 이름 (굵은 폰트, 흰색)
│  [등급 한국어명]            │  ← 서브타이틀 (반투명 흰색)
└─────────────────────────────┘
```

### 영역별 상세 규격

| 영역 | 위치 | 크기 | 내용 |
|------|------|------|------|
| **오행 아이콘** | 좌상단 8px 여백 | 28 × 28px | 오행 이모지 or 심볼 |
| **등급 뱃지** | 우상단 8px 여백 | 가변 높이 20px | N/R/SR/SSR/UR/SSS 텍스트 |
| **일러스트 영역** | 상단~하단정보 위까지 | 208 × 225px | 캐릭터 + 수호동물 + 배경 |
| **하단 정보 바** | 하단 70px | 208 × 70px | 카드명 + 등급명, 그라디언트 오버레이 |
| **테두리 프레임** | 전체 외곽 2px | 2px 두께 | 등급별 색상 |

### 하단 오버레이
- 배경: `linear-gradient(투명 → 검정 80%)` (위에서 아래로)
- 카드 이름: 흰색, 볼드, 14px (기준 사이즈)
- 등급명: 흰색 50% 투명도, 12px

## 8. 등급 시스템 시각 사양

각 등급은 **테두리 색상, 발광 효과, 파티클, 배경 분위기**로 차별화됩니다.

### 등급별 완전 시각 규격

---

#### N 등급 — 일반

| 속성 | 값 |
|------|-----|
| **메인 컬러** | #95a5a6 (쿨 그레이) |
| **테두리** | 1.5px, rgba(149, 165, 166, 0.4) |
| **글로우** | 없음 |
| **파티클** | 없음 |
| **배경 분위기** | 맑고 단순한 배경, 과도한 효과 없음 |
| **아우라** | 은은한 소프트 글로우 |
| **프레임 스타일** | 얇고 심플한 은색 테두리 |
| **일러스트 퀄리티** | 심플한 플랫 일러스트 |
| **캐릭터 표정** | 수줍음, 평온함, 기본 미소 |

**분위기 키워드**: 평화로운 아침, 소박함, 수수함, 첫 시작

## 10. 캐릭터 디자인 가이드

### 캐릭터 기본 방향
- **연령대 표현**: 아기 ~ 유아 (0~3세 느낌)
- **캐릭터화 수준**: 중간 수준 (완전 추상 아님, 완전 사실적 아님)
- **레퍼런스**: 사랑의 하츄핑 캐릭터 감성
- **얼굴**: 큰 눈, 통통한 볼살, 작은 코, 귀여운 미소

### 캐릭터 비율

```
머리 : 몸 = 약 1:1 ~ 1.2:1 (머리가 살짝 크게)

머리 크기: 전체 키의 약 45~50%
  ┌──────┐
  │ 큰 눈│  ← 눈 높이가 얼굴의 40% 위치 (아래쪽에 배치)
  │  얼굴│  ← 통통한 볼살, 작은 코
  └──────┘
     │
  ┌──┴──┐   ← 몸통 (약간 통통하게)
  │ 몸통│
  └──┬──┘
   ┌─┴─┐    ← 짧은 다리
   │   │
```

### 눈 표현
- **크기**: 얼굴의 약 25~30%를 차지하는 큰 눈
- **형태**: 동그란 눈, 반짝이는 하이라이트 (흰 점 2개 이상)
- **컬러**: 오행에 따라 변형 가능
  - 木: 초록빛 눈
  - 火: 황금색/주황빛 눈
  - 土: 갈색/앰버 눈
  - 金: 은빛/회색빛 눈
  - 水: 파란/보라빛 눈

### 표정 가이드라인

| 등급 | 표정 |
|------|------|
| N | 수줍은 미소, 평온한 눈빛 |
| R | 밝은 미소, 호기심 있는 눈빛 |
| SR | 신비로운 눈빛, 온화한 미소 |
| SSR | 따뜻한 빛남, 축복받은 표정 |
| UR | 압도적인 강렬함, 결의에 찬 눈빛 |
| SSS | 초월적 존재감, 온화함 속의 신성함 |

### 복장/장식 가이드라인

| 등급 | 의상 스타일 |
|------|------------|
| N | 소박한 유아복, 간단한 장식 |
| R | 귀여운 테마 의상, 소품 1~2개 |
| SR | 판타지 의상, 망토 또는 날개 |
| SSR | 화려한 왕족 의상, 관(冠) 또는 광환 |
| UR | 신화적 갑옷, 신기한 무기/소품 |
| SSS | 신화 속 신격화된 의상, 후광, 천상의 장식 |

### 오행별 소품/장식

| 오행 | 소품/장식 아이디어 |
|------|-------------------|
| 木 | 꽃잎 왕관, 나뭇잎 망토, 나무 요정 귀 |
| 火 | 불꽃 모자, 태양 문양 의상, 불사조 깃털 |

| 水 | 조개 왕관, 물방울 망토, 바다 보석 |

### 절대 금지 표현

- 찡그린 얼굴, 화난 표정
- 슬프거나 우울한 표정

- 실제 아기와 혼동될 수 있는 과도한 사실적 묘사

## 15. DO & DON'T — 아트 스타일 금기 및 권장

### DO (권장)

| 항목 | 설명 |
|------|------|
| 큰 반짝이는 눈 | 눈에 하이라이트 2~3개, 반짝임 표현 |
| 통통한 볼살 | 아기다운 부드러운 볼 |
| 따뜻한 미소 | 반달 눈, 올라간 입꼬리 |
| 파스텔 + 선명한 포인트 컬러 | 부드럽지만 오행 컬러가 뚜렷하게 |
| 부드러운 셀 쉐이딩 | 만화풍의 부드러운 명암 |
| 동물의 귀여운 버전 | 신화적 동물도 아기자기하게 |
| 빛 효과 | 등급에 맞는 적절한 글로우 |
| 화사한 배경 | 주제에 맞지만 캐릭터가 중심 |

### DON'T (금지)

| 항목 | 대신 할 것 |
|------|-----------|
| 화난/찡그린 표정 | 강인함은 눈빛으로, 표정은 온화하게 |
| 어두운/음울한 분위기 | 신비롭되 따뜻함 유지 |
| "예민", "까칠", "표독스러운" 표현 | 절대 금지 — 서비스 정체성 훼손 |
| 공격적인 자세 | 자신감 있는 포즈로 대체 |
| 지나치게 사실적인 아기 묘사 | 캐릭터화 유지 (캐릭터지, 실제 아기 아님) |
| 무기를 위협적으로 표현 | 소품으로만 (들고 있는 것, 공격 자세 아님) |
| 복잡하고 지저분한 배경 | 배경은 분위기용, 캐릭터가 주인공 |
| 등급별 일관성 깨는 색상 | 반드시 오행 팔레트 범위 안에서 |
| 이모지나 텍스트 포함 | 아트만 납품 |

## 16. 시리즈 일관성 유지 가이드

30종 + 추가 150종이 **하나의 시리즈처럼** 보여야 합니다.

### 고정 요소 (변경 불가)

| 요소 | 규칙 |
|------|------|
| **캐릭터 비율** | 머리:몸 = 1:1 ~ 1.2:1 고정 |
| **얼굴 구조** | 큰 눈, 동그란 얼굴, 작은 코 고정 |
| **눈 스타일** | 동그란 눈 + 하이라이트 2개 이상 고정 |
| **카드 세로형** | 항상 2:3 세로 비율 |
| **배경 어두움** | 카드는 어두운 배경 위에 표시되므로, 밝은 요소로 대비 |
| **라인 두께** | 아웃라인 2~3px (캔버스 208px 기준) |
| **음영 방식** | 부드러운 셀 쉐이딩 (에어브러시 x) |

### 가변 요소 (등급/오행에 따라 변경)

| 요소 | 변경 기준 |
|------|----------|
| 배경 컬러/분위기 | 오행별 팔레트 적용 |
| 테두리 컬러 | 등급별 컬러 적용 |
| 광효과 강도 | 등급 높을수록 강하게 |
| 의상 화려함 | 등급 높을수록 화려하게 |
| 수호 동물 크기/위상 | 등급에 따라 (소 → 대) |
| 아우라 색상 | 오행 + 등급 조합 |

### 스타일 시트 납품 요청

작업 완료 시 아래 스타일 시트 문서를 함께 납품해 주시면 AI 생성 시 활용합니다:

1. **컬러 팔레트 스와치** — 오행별 정확한 HEX 코드
2. **캐릭터 비율 도식** — 기준 비율 도면
3. **얼굴 가이드** — 눈/코/입 기준 위치
4. **라인 굵기** — 아웃라인/인라인 기준
5. **음영 레퍼런스** — 밝음/중간/어두움 3단계 기준
6. **배경 구성 예시** — 오행별 배경 레퍼런스

```

```

├── 01_character         // 캐릭터 레이어 그룹
│   ├── character_body
│   ├── character_face
│   └── character_accessories
├── 02_guardian          // 수호 동물 레이어 그룹

│   └── bg_base          // 기본 어두운 배경

    └── border

| **타로 카드** | 신비로운 분위기, 상징 요소, 세로형 구성 |
| **사랑의 하츄핑** | 귀여운 캐릭터 스타일, 아기자기한 감성 |

### 레퍼런스 적용 방향

- **포켓몬 카드**: 카드 구조와 등급 시스템을 참고하되, 일러스트는 동양풍 + 귀여운 캐릭터로 차별화
- **타로 카드**: 신비로운 분위기와 상징 요소 참고, 너무 어둡지 않게 밝고 따뜻하게 재해석
- **하츄핑**: 캐릭터 귀여움 수준의 기준점, 아기다운 표현의 레퍼런스

### 피해야 할 레퍼런스

- 무겁고 어두운 다크 판타지 (Dark Souls 류)
- 공포/호러 분위기의 카드 아트
- 성인 대상 게임의 섹시한 캐릭터 표현
- 서양식 판타지 (드래곤+기사+성) — 동양풍 유지

### 작업 순서 제안

1. **스타일 샘플 먼저** — 각 오행별 N등급 5종 (최소 효과) 먼저 확정
2. **SSS 1종** — 신화 등급 1종으로 최상위 퀄리티 기준 확정
3. **나머지 24종** — 확정된 스타일 기반으로 제작

### 컨펌 포인트

1. 캐릭터 기본형 컨펌 (비율, 얼굴 스타일)
2. 오행별 팔레트 컨펌 (색상 방향)
3. 등급별 효과 레벨 컨펌 (N vs SSS 차이)
4. 전체 30종 1차 컨펌
5. 수정 후 최종 납품

이 문서에서 불명확한 부분이 있으면 작업 시작 전에 반드시 확인 요청 부탁드립니다.

- 캐릭터 기본 비율 (머리 크기, 신체 비율)
- 수호 동물의 귀여움 수준 (얼마나 캐릭터화할지)
